# GEOMISIS Mobile (Flutter)

> Целевая платформа: iOS 14+ / Android 8+  
> Карта: Mapbox SDK + собственный стиль

## Папки
- `lib/` — основной код
- `assets/` — иконки, шрифты
- `test/` — widget-тесты

### Запуск

```bash
flutter pub get
flutter run
```

Приложение подключается к backend по адресу `http://localhost:8000/api/v1`.
